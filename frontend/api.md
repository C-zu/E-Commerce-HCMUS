# API USAGE

## PHONE

### SPECIFIC DATA

GET -----------------------------------------------------<br>
Get all data: http://localhost:3000/phone/specific_data/<br>
Get data using id: http://localhost:3000/phone/specific_data/{id}<br>
Get data with query: http://localhost:3000/phone/specific_data?{name=variable}<br>
POST -----------------------------------------------------<br>
New Data: http://localhost:3000/phone/specific_data/<br>
PUT -----------------------------------------------------<br>
Updata data using id: http://localhost:3000/phone/specific_data/{id}<br>
DELETE -----------------------------------------------------<br>
Delete data using id: http://localhost:3000/phone/specific_data/{id}<br>
